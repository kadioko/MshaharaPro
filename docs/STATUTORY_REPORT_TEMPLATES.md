# Statutory Report Template Notes

These templates are MVP scaffolds for payroll review and export. They are not a substitute for current TRA, NSSF, WCF, or SDL filing guidance.

## PAYE

Purpose: prepare monthly employee tax withholding review.

Current fields in app:

- Employer TIN
- Employee TIN
- Employee number
- Employee name
- Taxable pay
- PAYE withheld
- Payroll month

Before production filing, confirm:

- Current TRA PAYE template columns
- Required employer identifiers
- Employee identification requirements
- Rounding rules
- Submission file format

## NSSF

Purpose: prepare monthly pension contribution review.

Current fields in app:

- Employer NSSF number
- Employee NSSF number
- Employee number
- Employee name
- Gross pay
- Employee share
- Employer share
- Total contribution

Before production filing, confirm:

- Current NSSF schedule format
- Contribution caps or special cases, if any
- Accepted upload format
- Employer/employee split requirements

## WCF

Purpose: prepare Workers Compensation Fund contribution review.

Current fields in app:

- WCF registration number
- Employee number
- Employee name
- Gross earnings
- WCF contribution

Before production filing, confirm:

- Current WCF portal fields
- Sector-specific treatment
- Accepted gross earnings definition
- Required employer identifiers

## SDL

Purpose: prepare Skills and Development Levy review where applicable.

Current fields in app:

- Employer TIN
- Company name
- Employee count
- SDL applicable
- Total monthly emoluments
- SDL amount

Before production filing, confirm:

- Current employee-count threshold
- Current SDL rate
- Emoluments definition
- Exemptions or sector treatment
- Current TRA filing format

## Review Rule

Any template marked statutory must be reviewed by a qualified Tanzanian accountant or tax advisor before customer production use.
